Backup de Dados

Faça um backup completo dos seus dados financeiros para evitar perdas. O backup será salvo como um arquivo de banco de dados SQLite que você pode armazenar com segurança.

Dica: Recomendamos fazer backups regularmente, especialmente antes de importar grandes quantidades de dados.

Por que fazer backup?

  • Protege seus dados contra perda acidental
  • Permite transferir seus dados para outro computador
  • Mantém um histórico de suas finanças ao longo do tempo
  • Possibilita a restauração em caso de problemas com o aplicativo

Como restaurar um backup?

Para restaurar um backup, vá para a página Restaurar Backup e selecione o arquivo de backup que você deseja restaurar.

Atenção: A restauração de um backup substituirá todos os dados atuais. Certifique-se de fazer um backup dos seus dados atuais antes de restaurar.
