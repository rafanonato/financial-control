Restaurar Backup

Restaure seus dados financeiros a partir de um arquivo de backup.

Atenção: A restauração de um backup substituirá todos os dados atuais. Certifique-se de fazer um backup dos seus dados atuais antes de restaurar.

O que acontece durante a restauração?

Ao restaurar um backup:

  • Todos os dados atuais serão substituídos pelos dados do backup
  • Todas as suas categorias, transações e metas serão restauradas
  • A aplicação será reiniciada para aplicar as alterações

Não tem um backup?

Se você não tem um arquivo de backup, você pode:

  • Criar um novo backup agora na página Backup de Dados
  • Continuar usando o aplicativo com os dados atuais
  • Começar do zero importando novos dados
